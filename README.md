# Quote Generator


## Description
Using an API to build a random quote generator with Twitter integration.


## Demo
https://haticesurumlu.github.io/quote-generator/



### Logic

-  fetch api
-  async function
-  try / catch
-  css loader / spinner
-  twitter button => share quote as twitter tweet
-  encode uri component
